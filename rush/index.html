Game Project: Sweet Tomb


โปรเจค Sweet Tomb นำเสนอการพัฒนาเกมสยองขวัญโดยใช้ภาษา Java เป็นเครื่องมือหลัก ในการพัฒนาเกมดังกล่าวได้รับแรงบันดาลใจจากเกมแนวสยองขวัญเอาชีวิตรอด (Survival Horror) ที่มุ่งเน้น ไปที่การสร้างบรรยากาศที่น่าตื่นเต้น


กระบวนการพัฒนาเกมนี้ใช้เทคนิคการเขียนโปรแกรมเชิงวัตถุ (Object-Oriented Programming) และเครื่องมือที่เกี่ยวข้องเช่นการใช้ Thread, Java Swing สำหรับการแสดงผลกราฟิก และรวมไปถึงการออกแบบ กลไกการเล่น (Gameplay Mechanics) ที่ท้าทายและกระตุ้นการคิดวิเคราะห์ของผู้เล่น


เป้าหมายของโครงงานนี้คือการสร้างเกมที่ไม่เพียงแต่มอบประสบการณ์ความระทึกขวัญ แต่ยังทำให้ผู้พัฒนา นำความรู้ที่ได้จากรายวิชา (Object-Oriented Programming) มาประยุกต์ใช้ และสามารถสร้างสรรค์ผลงานจากสิ่งที่ได้เรียนรู้ได้
